FCC TimeStamp MicroService

What is this?

This is a microservice API project for Free Code Camp that will accept either a natural date format or a unix formatted date and return a json response with the date formatted as both. If the input is neither it will return null values.

Usage:

https://immense-brushlands-3354.herokuapp.com/January 10, 2015

https://immense-brushlands-3354.herokuapp.com/1420848000

Example Response:

{ natural: "January 10, 2015", unix: "1420848000" }
